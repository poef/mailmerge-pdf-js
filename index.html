Mail merge HTML to PDF POC

This version targets the currently used school-account activation templates. It validates strict enough to reject malformed tags, strips legacy ar:* attributes, fills {field} placeholders as text, and renders with pdf-lib.

    Supported subset in this POC

    Elements: document metadata, style blocks, headings, paragraphs, div/span text, strong/em/u, links, images, unordered/ordered lists, and simple table layouts used by email-style templates.

    Attributes: inline style, class, data-*, links, image sizes/alignment, and table/cell layout attributes. Legacy ar:* attributes are ignored and removed before rendering.

    The original Simant template is intentionally included as an expected failure because it has a malformed comment inside a <td> start tag. The fixed Simant template moves that comment outside the tag.
